HyTE Workflow

A stateful Agentic lifecycle for Telecom Hypothesis Testing powered by LangGraph and Gemini

Trigger
💡

User Hypothesis

User submits a natural language hypothesis. The Conversational Orchestrator initiates the graph state.

Grounding
🔍

RAG Retriever

Identifies relevant database tables and columns from the Enriched Data Dictionary to provide context.

Planning
📊

Methodology Agent

Designs a statistical analysis approach, defining metrics, filters, and success criteria for the test.

Logic
📑

Pseudo-code Agent

Translates abstract methodology into actionable logical steps, creating a clear blueprint for implementation.

Implementation
🐍

Code Gen Agent

Writes a production-quality Python script grounded in the retrieved metadata and following the pseudo-code logic.

Action
🚀

Execution Agent

Runs the code against the datalake. If it fails, a Self-Correction Loop triggers back to Code Gen.

Evaluation
🧪

Evaluation Agent

Analyzes the final artifacts and feedback to score performance and suggest prompt improvements or guardrails.
